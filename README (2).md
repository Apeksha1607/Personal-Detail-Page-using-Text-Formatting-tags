# 📌 Personal Profile Webpage

## 📖 Project Description

This project is a simple and responsive **Personal Profile Webpage** created using HTML and CSS.
It displays basic personal information such as name, age, course, hobbies, skills, and a short introduction in a stylish card layout.

The main objective of this project is to understand the use of:

* HTML text formatting tags
* Internal CSS styling
* Layout designing using Flexbox
* Creating a user-friendly interface

---

## 🛠️ Technologies Used

* **HTML5** – For structuring the webpage
* **CSS3** – For styling and layout design

---

## ✨ Features

* Attractive gradient background
* Card-based layout design
* Proper alignment using Flexbox
* Highlighted personal details
* Use of HTML formatting tags like:

  * `<mark>`
  * `<i>`
  * `<sup>`
  * `<sub>`
  * `<em>`
  * `<small>`
* Simple and clean user interface

---

## 👩‍💻 Personal Information Displayed

* Name
* Age
* Course
* Phone Number
* Hobbies
* Skills
* About Me section

---

## 🎯 Learning Outcome

Through this project, I learned how to:

* Design a basic webpage using HTML
* Apply CSS styling for better presentation
* Use different text formatting tags
* Create structured layouts using Flexbox
* Improve webpage appearance using colors and shadows

---

## 🚀 How to Run the Project

1. Download or copy the HTML file.
2. Save it with `.html` extension (e.g., `profile.html`).
3. Open the file using any web browser.

---

## 📍 Output

The webpage displays a personal profile in a centered card format with a gradient background and formatted text.
